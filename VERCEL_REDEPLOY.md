# Как задеплоить изменения на Vercel

## Способ 1: Автоматический деплой (если настроен)

Если Vercel подключен к вашему GitHub репозиторию с автодеплоем:

1. **Подождите 1-2 минуты** - Vercel автоматически обнаружит изменения в GitHub
2. Зайдите на https://vercel.com
3. Откройте ваш проект
4. Проверьте вкладку **"Deployments"** - должен появиться новый деплой
5. Дождитесь завершения деплоя (статус "Ready")

---

## Способ 2: Ручной деплой через Vercel Dashboard

1. Зайдите на https://vercel.com
2. Войдите в свой аккаунт
3. Откройте ваш проект (delivery)
4. Нажмите на кнопку **"Deployments"** (или "Deployments" в меню)
5. Нажмите на **"..."** (три точки) рядом с последним деплоем
6. Выберите **"Redeploy"**
7. Подтвердите передеплой

---

## Способ 3: Через Vercel CLI (если установлен)

```bash
cd frontend
vercel --prod
```

---

## Проверка после деплоя

После завершения деплоя:

1. Откройте ваш сайт (Vercel URL)
2. **Очистите кеш браузера** (Ctrl+Shift+R или Cmd+Shift+R)
3. Проверьте:
   - Карусель показывает категории (не рестораны)
   - При клике на категорию показываются блюда из всех ресторанов
   - В админ-панели есть страница "Karusel" для управления категориями

---

## Если изменения не появились

1. **Проверьте, что деплой завершился успешно:**
   - В Vercel Dashboard → Deployments → проверьте статус
   - Должен быть статус "Ready" (зеленый)

2. **Очистите кеш браузера:**
   - Chrome/Edge: Ctrl+Shift+Delete → выберите "Cached images and files"
   - Или используйте режим инкогнито

3. **Проверьте URL:**
   - Убедитесь, что открываете правильный URL (production, не preview)

4. **Проверьте логи деплоя:**
   - В Vercel Dashboard → Deployments → выберите последний деплой → "View Build Logs"
   - Убедитесь, что нет ошибок

---

## Быстрая проверка

Откройте консоль браузера (F12) и проверьте:
- Нет ли ошибок JavaScript
- Загружаются ли новые файлы (Network tab)

---

## Если всё ещё не работает

1. Проверьте, что изменения действительно в GitHub:
   - https://github.com/elyorka22/delivery
   - Проверьте последний коммит: "Change carousel from restaurants to categories..."

2. Проверьте настройки Vercel:
   - Root Directory должен быть: `frontend`
   - Build Command: `npm run build`
   - Output Directory: `dist`

3. Попробуйте создать новый деплой:
   - В Vercel Dashboard → Settings → Git → Disconnect
   - Затем снова Connect и выберите репозиторий
