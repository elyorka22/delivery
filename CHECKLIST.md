### Что нужно сделать перед деплоем

1. **Создать .env файлы:**
   - Backend: `DATABASE_URL`, `JWT_SECRET`, `PORT`, `FRONTEND_URL`
   - Frontend: `VITE_API_URL` (опционально)

2. **Создать супер-админа:**
   - После деплоя создать через базу данных или скрипт

3. **Настроить продакшен:**
   - Переключить на PostgreSQL (рекомендуется)
   - Изменить JWT_SECRET на безопасный
   - Настроить HTTPS
   - Настроить домены

4. **Собрать проект:**
   ```bash
   # Backend
   cd backend
   npm run build
   
   # Frontend
   cd frontend
   npm run build
   ```
